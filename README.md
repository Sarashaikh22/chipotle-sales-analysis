# Chipotle Sales & Product Performance Analysis

## 📊 Project Overview
This project analyzes Chipotle order data to uncover insights into sales, customer demand, and pricing strategy.

## 🛠 Tools Used
- SQL (SQLite)
- Power BI

## 🔍 Key Insights
- Chicken Bowl is the top-selling and highest revenue-generating item
- Chicken Salad Bowl is the most expensive item (premium category)
- A few bulk orders contribute significantly to total revenue
- Demand and pricing together drive revenue performance
